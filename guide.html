What does that mean?

Foundation allows you to create responsive, multi-device layouts with a grid system. Adding an element of class .row would creates a horizontal block to contain columns. Next, add columns (you guessed it, of class .column! Or .columns -- it's the same thing). To specify the width, use the following classes: .small-#, .medium-#, and .large-#. The small, medium, and large essentially specify how the column would collapse (try playing around with the window size!). This paragraph and header is in an element called a callout.
