What does that mean?

Foundation allows you to create responsive, multi-device layouts with a grid system. Adding an element of class .row would creates a horizontal block to contain columns. Next, add columns (you guessed it, of class .column! Or .columns -- it's the same thing). To specify the width, use the following classes: .small-#, .medium-#, and .large-#. The small, medium, and large essentially specify how the column would collapse. Smalls tend to stay together on the same row when resized, where as the large and mediums just collapse under each other. This is very crucial if you were to visit the website on a mobile device. How to Use Foundation

We started off with a lot of stuff that's pretty specific, but we never really talked about what Foundation is or how to use it in general.

What is Foundation? It's a CSS framework, which, according to Wikipedia, is a pre-prepared library that allows for "easier, more standards-compliant styling of web pages using the Cascading Style Sheets language... CSS frameworks are usually incorporated as external .css sheets referenced in the HTML head." Here's the link to the Wikipedia article, for reference. In other words, it makes your content purrrttyy!

Anywho, that means it's not voodoo, so it's not difficult to use! Start by going to the Foundation page. You can download the complete version to get all that they offer, or you can fill out the form to customize your framework, or what features you want. That way you don't have any unnecessary bulk -- if you only want a CSS grid and buttons, then checking those and nothing else will give you that and nothing else.

After that, you can pretty much just use it how you like! The kitchen sink is really useful because it shows you all of the features they offer in one page (and the code). Y'all know how classes work, so you're just going to use them like so :)
